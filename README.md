# Aula Invertida e IA en CAFD

Portal web institucional para el proyecto de innovación docente **Aula Invertida e IA en CAFD** del Grado en Ciencias de la Actividad Física y del Deporte de la Universidad de Zaragoza.

## Tecnología

- Next.js con App Router
- React
- TypeScript
- Tailwind CSS

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

La aplicación está preparada para desplegarse en Vercel como proyecto Next.js sin configuración adicional de directorio de salida.
